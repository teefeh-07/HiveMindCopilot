HiveMind Copilot Example Projects

This page showcases example projects demonstrating how to use HiveMind Copilot with Hedera's blockchain services. Each example includes code snippets and explanations to help you get started.

Token Service Example

Create and manage fungible and non-fungible tokens on the Hedera network.

Key Features:

  • Token creation with custom properties
  • Token transfers
  • Token information queries

View Token Service Example

Smart Contract Analysis

Analyze smart contracts for security vulnerabilities and optimization opportunities.

Key Features:

  • Security vulnerability detection
  • Gas optimization suggestions
  • Code quality analysis

View Smart Contract Analysis Example

Consensus Service Implementation

Implement Hedera Consensus Service for decentralized messaging and event logging.

Key Features:

  • Topic creation
  • Message submission
  • Message subscription

View Consensus Service Example

Note: These examples are designed to work with the HiveMind Copilot VS Code extension. Make sure you have installed the extension from the file hivemind-clean/hivemind-copilot-0.3.0.vsix and configured it to connect to the backend API at http://localhost:8005.

Getting Started

To run these examples:

  1. Install the VS Code extension
  2. Configure the extension to connect to the backend API
  3. Open the example project in VS Code
  4. Follow the instructions in the README.md file
