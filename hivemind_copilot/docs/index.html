HiveMind Copilot Documentation

Introduction

HiveMind Copilot is an AI-powered development environment that integrates decentralized AI agents with Hedera's enterprise blockchain. It provides developers with intelligent assistance for building applications on the Hedera network.

Getting Started

To use HiveMind Copilot, you need to:

  1. Install the VS Code extension from the file hivemind-clean/hivemind-copilot-0.3.0.vsix
  2. Configure the extension to connect to the backend API at http://localhost:8005
  3. Start coding with Hedera!

Features

  • Smart Contract Analysis - Get insights into your smart contracts
  • Token Service Integration - Easily create and manage tokens on Hedera
  • Consensus Service - Implement Hedera Consensus Service with AI assistance
  • Code Completion - Get intelligent code suggestions for Hedera SDK

Configuration

To configure the HiveMind Copilot extension:

  1. Open VS Code settings (Ctrl+, or Cmd+,)
  2. Search for 'HiveMind'
  3. Set 'HiveMind: Backend URL' to 'http://localhost:8005'

Note: For this hackathon evaluation version, we've pre-configured the environment with Docker Compose for easy setup.

Example Projects
